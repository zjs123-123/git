## 总体视觉风格定位

**关键词**：乡村纪录片 × AI任务系统 × 青年行动感

- ❌ 不要：二次元、赛博朋克、廉价蓝光科技风、游戏UI过重
- ✅ 要：山区真实感、半纪录片质感、温暖接地气、AI辅助感自然不突兀

**配色建议**：大地色系（暖棕、稻黄、山脉灰绿）+ 少量科技蓝灰作为系统提示色

## 【P5】任务推送页 —— 为什么群众不愿意参与建设？

**对应教学环节**：（二）任务一启动 —— AI系统弹窗 + 村民反馈信息流

**页面内容**：
- AI系统警告图标（红色闪烁动画）
- 【当前任务状态】乡村共创行动推进失败
- 三条原因：群众不愿配合调研 / 村民对实践团缺乏信任 / 产业方案推进停滞
- 底部"村民反馈信息流"（以聊天弹窗形式呈现）

**AI工具与制作方式**：

### 1. 村民反馈信息流 —— 豆包生图做头像 + PPT动画

| 项目 | 说明 |
|------|------|
| **工具** | 豆包 生成村民头像 |
| **提示词** | `中国农村老年男性正面肖像，60岁左右，黝黑皮肤，皱纹清晰，穿着朴素深色衣服，生活照质感，不要过度美化` |

### 2. 信息流动画 —— PPT触发器实现

在PPT中做出"弹窗依次弹出"效果：
- 第1条弹窗："以前也有人来调研，最后没结果。"（带村民头像）
- 第2条弹窗："你们年轻人懂农村吗？"
- 第3条弹窗："产业项目以前赔过，干部说得好听。"
- 第4条弹窗："最后还不是我们自己承担风险？"

**制作方法**：每条弹窗设置"飞入"动画，触发方式设为"上一动画之后"，间隔0.5秒

### 3. AI警告图标

| 项目 | 说明 |
|------|------|
| **工具** | Canva → 搜索"警告图标"→ 修改颜色为红色 → 设置脉冲动画 |
| **或** | PPT自带图标库 → 插入图标 → 搜索"警告" |

**预期效果**：信息流形式让学生的直观感受是"真实村民在说话"，比阅读大段文字更有代入感。

## 【P6】关键人物解锁页 —— 黄文秀基层工作档案

**对应教学环节**：（三）关键人物解锁 —— 系统提示 + 学生选择解锁

**页面内容**：
- AI系统检测框：【系统检测】群众工作推进受阻。是否解锁"基层工作档案"？
- 两个大按钮：【是 / 否】
- 背景：模糊的村庄夜景

**AI工具与制作方式**：

### 1. 背景图（豆包生图）

| 项目 | 说明 |
|------|------|
| **工具** | 豆包 |
| **提示词** | `中国西南山村夜晚，雨后泥泞路面反射远处窗户灯光，氛围感夜景，电影质感，画面略微模糊作为背景，色调偏暗暖` |

### 2. PPT互动按钮（PPT触发器实现）

**这不是AI做的，但非常关键**——要让课堂有"系统操作感"：
- 制作两个大按钮"是"和"否"（用PPT形状+文字即可）
- 给按钮添加点击触发的超链接/动画效果
- 点击"是"→ 跳转到P7（场景进入页）
- 点击"否"→ 弹出提示"任务继续受阻"，最终还是引导到"是"

**制作方法**：
1. 插入 → 形状 → 圆角矩形 → 设置颜色为深蓝灰
2. 右键 → 超链接 → 本文档中的位置 → 选择P7
3. 添加悬停效果（形状格式 → 形状效果 → 阴影）

**预期效果**：学生喊出"是"的那一刻，课堂节奏由学生自己推动，参与感立刻建立。

- 三个选项按钮：A / B / C
- 底部：系统提示"请选择你的行动"

**AI工具与制作方式**：

### 1. 紧闭木门图（豆包生图）

| 项目 | 说明 |
|------|------|
| **工具** | 豆包 |
| **提示词** | `中国西南农村老旧木门特写，门紧闭，门缝透出暖黄色灯光，门上贴着褪色的对联，傍晚氛围，真实农村摄影，竖构图` |

### 2. 选项按钮 + 系统反馈（PPT触发器动画）

**这是PPT互动设计的核心**——三种选择，三种不同反馈：

| 选项 | 内容 | 点击后动画 |
|------|------|-----------|
| A | 强调这是政策要求，希望村民配合 | 弹出红色反馈："村民情绪更加抵触。" |
| B | 留下调查表，改天再来 | 弹出橙色反馈："信息收集效率较低，群众关系没有改善。" |
| C | 主动帮助村民做事，再寻找交流机会 | 弹出绿色反馈 → 自动跳转到P9 |

**制作方法**：
1. 为A/B/C各做一个圆角矩形按钮
2. 为每个按钮设置"单击时"触发器，分别显示对应的反馈文本框
3. C选项额外设置：反馈出现2秒后自动跳转P9

**这一步不需要AI**，用PPT自带触发器即可实现，但效果非常关键——学生真正在选择中体验"基层工作决策"。

# 五、PPT创新亮点总结（答辩/申报时可重点表述）

1. **AI智能体嵌入课件**（P11/P23）：用扣子Coze创建"黄文秀基层工作档案"和"竺可桢地理智库"两个AI智能体，学生可在课堂上与AI进行真实对话互动，而非观看预录内容。

2. **自定义任务提交与分析平台**（P16/P28）：用简道云搭建"乡村共创任务平台"，学生课后在线提交方案，系统自动汇总分析，AI辅助评估反馈，形成完整的"教学-提交-分析-反馈"闭环。

3. **AI角色驱动的情境决策系统**：学生不是被动观看AI生成的内容，而是在每个关键节点做出选择（A/B/C），AI系统给予不同反馈，实现"决策-反馈"的互动学习。

4. **双智囊AI架构**（P27）：将"思政价值引领"（黄文秀）与"地理科学分析"（竺可桢）两个AI智能体并置，形象化呈现跨学科融合的设计理念。

5. **AI全流程赋能**：从情境生成（AI视频）→ 角色互动（AI智能体）→ 知识分析（AI智库）→ 方案提交（在线平台）→ 成果评估（AI分析），AI不是点缀，而是贯穿教学全过程的学习支架。

6. **"真实感"美学取向**：区别于常见的"科幻AI课堂"风格，本作品坚持"乡村纪录片×AI任务系统"的美学取向，用AI还原真实中国乡村质感，让技术服务于人文叙事。
